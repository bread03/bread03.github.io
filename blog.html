Diving into AI and Deep Learning

Date: 28/02/2024

I have always wanted to delve into AI and machine learning but was always daunted by the task of it but luckily I came across a great learning resource by fast.ai.

Practical Deep learning course by fast.ai

So far I am up to lesson 2 of the course, and hope to deploy a my own macine learning project. So far in following the course, I have experimented with fastai and Pytorch, creating and training a simple data classification model for cats and dogs. Although quite simple, this is my first dive into deep learning.

What I learnt:
  • The basics of deep learning
  • How to set up and train a simple model using fastai
  • Data blocks and how to fine tune the model
What I need to learn:
  • How to improve error rate
  • Training a model for other types of media
  • And a bunch more stuff, obviously...
